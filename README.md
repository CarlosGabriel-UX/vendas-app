# VendasApp - E-commerce Premium

Projeto de E-commerce desenvolvido com Next.js, Supabase e Tailwind CSS.

## Funcionalidades
- Design Moderno (Lovable UI)
- Sistema de Login/Cadastro
- Carrinho de Compras
- Checkout com Integração de Pagamento
- Painel Admin Seguro
- Dark Mode

## Deploy
Deploy automático via Vercel.
